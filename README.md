#C++ File Cryptosystem
A low level console-based file processing application built in C++.
This application demonstrates the core mechanics of symmetric-key cryptrography and secure file I/O.

## Execution Flow-
the interface provides three primary operational modes- 
**Add & Encrypt:** This helps in encrypting the plain-text using master key and secret-labels.
**Decrypt and Retrieve:** This helps in verifying credentials using the master verification key and retrieving the specific records via secret labels.
**Exit & Clear:** This safely terminates execution and helps in clearing the system data channels.
